# Biblioteca — Protótipo Dark com Login (React + Tailwind)

Protótipo com login simulado (front-end).

## Como usar

1. Instale Node.js (18+ recomendado)
2. Extraia o ZIP
3. No terminal, dentro da pasta do projeto:
   ```bash
   npm install
   npm run dev
   ```
4. Abra http://localhost:5173

## Observações
- Autenticação é simulada no front-end (useState)
- Perfis disponíveis: Leitor, Bibliotecário, Administrador
- Projeto front-end pronto para integrar com backend posteriormente
